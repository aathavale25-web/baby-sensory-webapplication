# Baby Sensory World - Documentation

Welcome to the Baby Sensory World documentation! This web application provides age-appropriate sensory experiences for babies aged 4-12 months.

## ðŸŒŸ Features

- **Age-Adaptive Experience**: Automatically adjusts complexity based on baby's age (4-6mo, 7-9mo, 10-12mo)
- **Personalized Interaction**: Uses baby's name in celebrations and feedback
- **Multiple Themes**: Ocean, Space, Garden, Rainbow, Animals, Shapes, Clouds, and Contrast World
- **Progress Tracking**: Records touches, streaks, and milestones
- **Session Analytics**: Tracks play sessions with detailed statistics
- **Supabase Integration**: Cloud storage for profiles and sessions
